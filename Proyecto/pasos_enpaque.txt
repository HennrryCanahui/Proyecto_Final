2. Crea un entorno virtual 
    python -m venv virtual              (Creas)
    virtual\Scripts\activate            (Activar)

    deactivate                          (Desactivar)


3. Intalas todas las librerias que utilizaras en el proyecto    
    este caso:
    pip install flet                    (Parte grafica)
    pip install pyinstaller             (Empaquetador)
    pip install mysql-connector-python  (Sql)

4. Comando de Empaquetador
    pyinstaller --onefile main.py
